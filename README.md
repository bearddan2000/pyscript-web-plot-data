# pyscript-web-plot-data

## Description
A demo of pyscript plotting downloaded dataset.

## Tech stack
- pyscript

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`
- [Available at](http://localhost)

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
